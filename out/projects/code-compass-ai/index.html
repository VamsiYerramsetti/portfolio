Code Compass AI

The secure assistant for Databricks notebooks with deep cross-notebook understanding across your entire repository that helps teams understand, search, and optimize code with context-aware, persona-based explanations.

GenAIRAGDatabricksAzure OpenAILangChainPythonChroma Vector DBSemantic SearchDAPT25 vitality innovation winner

Narrative

Code Compass AI was developed to address the challenge of understanding and optimizing large Databricks notebook repositories. Teams struggled with fragmented code, lack of context, and slow onboarding. The project aimed to provide a secure, context-aware assistant that could semantically search, explain, and connect code across notebooks, improving productivity and code quality without exposing sensitive production data.

Approach

  • Framed the problem around repository-wide semantic search and persona-based code explanations
  • Designed a pipeline using LangChain, Azure OpenAI, and Chroma Vector DB for deep retrieval and context linking
  • Developed privacy-first architecture—no access to production data, only code and metadata
  • Implemented Markdown-to-HTML rendering for rich, copyable code blocks
  • Validated with internal users and iterated on feedback for usability and security

Outcome

Code Compass AI delivered repository-wide semantic search, context-aware explanations, and persona-based guidance. It improved onboarding speed, code discoverability, and developer productivity. The privacy-first design ensured compliance, and the project was awarded “most viable innovation” in an internal challenge.

Highlights

  • • Semantic search and repository-wide retrieval
  • • Persona-based responses for all skill levels
  • • Privacy-first design—no production data access
  • • Markdown-to-HTML rendering with copyable code blocks
  • • Awarded most viable innovation (internal challenge)
